== Description ==

A quicklook into WP Classifieds Listings

= User features =
    ★ Create and manage classified listings using the traditional WordPress interface.
    ★ Users can post their  listings as wanted or offered.
    ★ Create image galleries with classified listings.
    ★ Specify location/contact details with  classified listings.
    ★ Option to post premium classified listings.
    ★ Registered users can see their own payment details .

= Admin features =
    ★ Manage classified listings as custom post type.
    ★ Allow admin to preview listing before going live.
    ★ Configure listing expiry days.
    ★ Configure listing display order.
    ★ Admin can set image gallery as Wordpress default or external gallery plugin.
    ★ Admin can define custom fields for classifieds post type.
    ★ Configure Paypal settings and currency format settings.
    ★ Admin can set premium listing price and period configurations.
    ★ Admin can configure custom or existing roles for classifieds users.
    ★ Admin can set custom login management pages.
    ★ Admin can set custom pages for classifieds related operations.
    ★ Admin can set default  country for the classifieds site.
    ★ Admin can  edit or add  cities.
    ★ Admin can edit or add  regions.
    ★ Admin can view all payment details.
    ★ Classifieds listings can be edited, deleted, approved, made premium or free.

 = Front end features =
    ★ Classifieds listing based on regions and cities.
    ★ Browse ads by category.
    ★ Search ads by listing title or description.
    ★ Premium ads display at the top of each category page.
    ★ Recent ads are shown in home page.


 WP Classifieds Listings is a classifieds listing plugin for adding classifieds functionality to your WordPress site. This is a fully functional classifieds plugin with most essential functionalities included.

== Installation ==

1. Extract `wp-classifieds-listings.zip` to your `/wp-content/plugins/` directory.
2. In the admin panel under plugins activate WP Classifieds Listings.
3. You can configure the settings from WP Classifieds Listings menu.
4. It should now be completely set up and functional.

If you need any further help, you may contact our [support desk](http://xyzscripts.com/support/ "XYZScripts Support").
